# Portfolio
Enter Portfolio, this is my way of decently representing my skills and projects. Built using basic tools. 

## Tech Stack
- Frontend: Traditional(HTML, CSS, JS)
- Backend: Flask
- Database: MongoDB
- Hosting(Frontend): Vercel
- Hosting(Backend): Render

## Features
- A small know about me, and timeline
- A showcase of the skills I possess, with the projects I have currently created
- A page which contains links to my social media handles
- All these works on tablets, phones and laptops
